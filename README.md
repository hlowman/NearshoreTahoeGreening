# NearshoreTahoeGreening
Repository containing scripts and data to support the following manuscript:

*Lowman, H.E., K. Loria, L. Katona, J. Krause, S. Chandra, S. MacIntyre, J.M. Melack, R. Naranjo, J.R. Blazczak. Influences of biological, physical, and watershed drivers on daily dissolved oxygen dynamics in nearshore regions of an oligotrophic mountain lake. In Prep.*

For additional support or information regarding this project, please contact Heili at heili.lowman@duke.edu.
